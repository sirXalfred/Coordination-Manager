# Start Guide

This repository is a clean public snapshot of Coordination Manager.

## Project Layout

- `Code/` holds the monorepo source.
- `docs/` holds public documentation and proposals.
- `scripts/` holds repo automation.

## Notes

- Keep private-only files out of public syncs.
- Follow the scripts in `package.json` for development and tests.
- See `README.md` for the normal setup and contribution flow.
